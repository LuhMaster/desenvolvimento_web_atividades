 Crianças do Futuro

Site institucional fictício de uma ONG chamada *Crianças do Futuro*, desenvolvido para fins de estudo e demonstração.  
O projeto apresenta a proposta da ONG, seus projetos sociais e um formulário de contato, com interações visuais em JavaScript.

Sobre o projeto
O Crianças do Futuro é um site de uma ONG fictícia, criado com o objetivo de praticar o desenvolvimento web front-end utilizando HTML, CSS e JavaScript.  
O site possui design responsivo, interatividade e foco em uma experiência leve e acessível para o usuário.

Tecnologias utilizadas
- HTML5 — estrutura do conteúdo  
- CSS3 — estilização, responsividade e layout  
- JavaScript (ES6) — interatividade e animações  
- Minificação — otimização dos arquivos CSS e JS para melhor desempenho  

Funcionalidades
  Formulário de contato funcional (exemplo de envio ou simulação de envio)  
  Descrição dos projetos da ONG, apresentando suas ações e metas  
  Banner interativo com destaque para causas e campanhas  
  Animação dinâmica: uma bola que se move aleatoriamente pela tela durante a navegação  
  Layout responsivo, adaptando-se a diferentes tamanhos de tela
